# Code Workspace Test

This is a test workspace to demonstrate Phase 2: Read-Only File Operations.

## Features

- **FileBrowser**: Browse files using Continue's walkDir patterns
- **Monaco Editor**: View files with syntax highlighting
- **Security**: Path validation from Continue + risk assessment from Jarvis
- **Line Numbers**: Codex's L123: format for easy reference

## Integration Sources

- **Continue**: File operations (walkDir, shouldIgnore, readFile)
- **Codex**: Line-numbered reading, indentation-aware parsing
- **Jarvis**: Permission layer with risk assessment
- **Big Query**: Template orchestrator (coming in Phase 11)

Ready for Phase 3: Write operations with diff preview!
